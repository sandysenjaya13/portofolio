Website Mantapnyoss

Website seputar game Point Blank yang menyajikan informasi tips/trik dan hiburan.

Website ini sebagai sarana penulis untuk menyalurkan hobinya, yaitu berbagi tips/trik dalam bermain game Point Blank.

Mantapnyoss sudah diterima menjadi partner Google AdSense dan dapat menayangkan iklan untuk mendapatkan income bagi penulis.

Dibuat menggunakan Blogger dengan template Viomagz. Saya redesign template ini untuk menyesuaikan tema game FPS. Salah satu penerapannya adalah penggunaan latar gelap/dark mode.
